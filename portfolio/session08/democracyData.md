## Gapminder datasets: Democracy Index

Source data 'Democracy Score' from [Gapminder](https://www.gapminder.org/data/). Data derived from [Polity](https://www.systemicpeace.org/polityproject.html). Each index value ranges from -10 to 10 and is calculated by subtracting the 'autocracy score' from the 'democracy score'.

The first few rows look like this:

^^^elm {m=(tableSummary 8 democracyTableFull)}^^^

Many of the cells are empty, especially for earlier years, so we can filter the table to show just those countries with data since 2010:

```elm {l}
democracyTable : Table
democracyTable =
    democracyTableFull
        |> filterColumns (\colName -> colName >= "2010")
        |> filterRows "2010" (\val -> val /= "")
        |> filterRows "2011" (\val -> val /= "")
```

^^^elm {m=(tableSummary 8 democracyTable)}^^^

## Joining with the Location Table

As with the bad teeth example, we can join this table to the table of locations, matching only those with matching country names:

```elm {l}
democracyGeoTable : Table
democracyGeoTable =
    innerJoin "CountryName"
        ( democracyTable, "country" )
        ( locationLookup, "CountryName" )
```

^^^elm {m=(tableSummary 8 democracyGeoTable)}^^^

### Values in democracy table but not in country lookup

Using the [leftDiff](https://package.elm-lang.org/packages/gicentre/tidy/latest/Tidy#leftDiff) table function we can check for any values in the democracy table that are not listed in the location table.

```elm {l}
displayLeftDiff : Table
displayLeftDiff =
    leftDiff ( democracyTable, "country" ) ( locationLookup, "CountryName" )
```
